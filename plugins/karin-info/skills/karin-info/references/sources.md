# Karin. 情報源リファレンス

## 情報源一覧

### 1. WebSearch

最新情報の取得に最も有効。

推奨クエリ:
- `"Karin." ユニバーサルミュージック [キーワード]`
- `"Karin." シンガーソングライター [キーワード]`
- `site:natalie.mu "Karin."`
- `site:universal-music.co.jp karin`
- `site:karin-official.com [キーワード]`

混同注意: 「Karin.」は一般的な名前のため、KARA（K-pop）等が混入しやすい。URLドメインで必ずフィルタリングすること。

### 2. 公式サイト sitemap（WebFetch可能）

URL: `https://www.karin-official.com/`
プラットフォーム: Wix（JSレンダリング → 通常ページはWebFetch不可）

読み取り可能なsitemap:

| sitemap URL | 内容 |
|-------------|------|
| `https://www.karin-official.com/sitemap.xml` | サイトマップインデックス |
| `https://www.karin-official.com/event-pages-sitemap.xml` | ライブイベント一覧（URLに日付+会場名） |
| `https://www.karin-official.com/pages-sitemap.xml` | 全ページ一覧（日付ページ、ニュース含む） |

URLパターンから日付・会場を抽出可能。`lastmod` で更新日も確認できる。

- 特徴: 公式レーベルページ。リリース情報が正確
